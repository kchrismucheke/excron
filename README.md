# Excron

A cron-like library to run jobs in Elixir.

  - Easy config
  - Tiny and readable
  - Cluster support with each job running in only one node

## Installation
````
def deps.do 
[
    {:excron, [github: "kchrismucheke/excron"]}
]
end
````
